## Node types

| Type | ID format | Key fields |
|---|---|---|
| Signal | `signal-YYYY-MM-DD-slug` | title, date, source_type, cla_layer, protocol_stage |
| Concept | `concept-kebab-name` | label, signal_count, first_seen |
| Practitioner | `practitioner-kebab-name` | label, type (person/org/collective/institution), signal_count |
| Scene | `scene-kebab-name` | label, geography, signal_count |
| Thread | `thread-kebab-slug` | label (question), status (open/closed), signal_count |

## Edge types

| Edge | Meaning | Weight |
|---|---|---|
| concept ↔ concept | Co-occurrence in same signal | Increments per co-occurrence |
| practitioner ↔ concept | Practitioner mentioned in signal containing concept | Increments per co-occurrence |
| scene ↔ concept | Scene tagged in signal containing concept | Increments per co-occurrence |
| thread ← signal | Signal is evidence for thread | 1 |

## The six-stage protocol

```
SENSE → QUERY → SPECULATE → REACT → COLLABORATE → EXPERIMENT
```

| Stage | What happens |
|---|---|
| **SENSE** | Raw signals captured — conversations, observations, transcripts, links |
| **QUERY** | Patterns and tensions surfaced across the corpus |
| **SPECULATE** | Language forms — movements named, frameworks proposed |
| **REACT** | Provocations sent back to the field; dialogue returns as new signals |
| **COLLABORATE** | Field becomes co-builder; relational graph activates |
| **EXPERIMENT** | Intelligence returned as briefs, exhibitions, capital frameworks |

*We are here: SENSE.* All current signals are at the sensing stage.

## CLA layers

| Layer | What it captures | Signal type |
|---|---|---|
| **Litany** (L1) | Observable facts, metrics, events | Quantitative, web data |
| **Social** (L2) | Structural causes, institutional patterns, networks | Secondary sources |
| **Discourse** (L3) | Concepts, vocabularies, theoretical frames | Practitioner interviews |
| **Myth** (L4) | Lived experience, values, cultural meaning | Deep conversations, installations |

Quantitative signals are primarily L1-L2. Qualitative practitioner conversations are primarily L2-L4. **The semantic gap exists because most systems only process L1-L2.** A(DAI)'s value is in the L3-L4 layer — where the meaning lives.

## Intelligence tiers

| Tier | Source |
|---|---|
| **Primary** | Direct practitioner knowledge, interviews, first-hand observation |
| **Secondary** | Credible indirect sources, contributed links, team observations |
| **Tertiary** | Articles, reviews, institutional announcements, scout-sourced web content |

## Confidence levels

| Level | Meaning |
|---|---|
| **high** | Verified source, direct contribution, named practitioner |
| **medium** | Credible secondary source, indirect contribution |
| **low** | Unverified, or algorithmically inferred |
| **speculative** | Explicitly marked inference or projection |
